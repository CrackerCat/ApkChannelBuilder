# ApkChannelBuilder
<img width="100" src="https://github.com/baoyongzhang/ApkChannelBuilder/blob/master/Screenshots/icon.png?raw=true"/>  
一个 Mac App，用于 Apk 多渠道打包。  

* 使用美团的方案，在 META-INF 目录写入空文件
* 支持 AES 加密，CBC 和 ECB 两种模式
* 可以自定义前缀
